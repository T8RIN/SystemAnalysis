# Отчёт по лабораторной работе №5

## Тема

Обнаружение людей на изображении, видео и в веб-камере с использованием
фреймворка `Ultralytics YOLO11`.

## Цель работы

Изучить возможности `Ultralytics YOLO11` для задач обнаружения объектов и
разработать программу, которая:

- загружает предобученную модель;
- выполняет детекцию людей или лиц;
- выделяет найденные объекты рамками;
- выводит количество обнаруженных объектов на экран и в консоль;
- поддерживает изображение, видеофайл и веб-камеру.

## Краткое описание реализации

В качестве базовой модели используется `yolo11n.pt`, обученная на наборе данных
`COCO`. Программа написана на `Python` и использует:

- `ultralytics` для загрузки модели и инференса;
- `opencv-python` для чтения изображений и видеопотока, отрисовки рамок и вывода окна;
- `numpy` для представления кадров в виде массивов.

Основной сценарий расположен в `main.py`. Он поддерживает следующие режимы:

- обработка одного изображения;
- обработка видеофайла покадрово;
- обработка потока с веб-камеры в реальном времени.

По умолчанию программа фильтрует результаты по классу `person`. Для каждого
найденного объекта отображаются:

- ограничивающая рамка;
- подпись с названием класса;
- уверенность модели (`confidence`).

В левом верхнем углу кадра выводится общее количество найденных объектов
выбранного класса.

## Ответы на контрольные вопросы

1. Какие задачи компьютерного зрения поддерживает YOLO11?

`YOLO11` поддерживает обнаружение объектов, сегментацию экземпляров,
классификацию изображений, оценку позы и обнаружение ориентированных объектов.

2. В чём разница между режимами `predict`, `train`, `export`?

- `predict` — выполняет распознавание на новых изображениях, видео или потоке;
- `train` — обучает или дообучает модель на наборе данных;
- `export` — преобразует обученную модель в формат для развёртывания, например `ONNX`, `TensorRT`, `OpenVINO`.

3. Какие классы содержит набор данных `COCO`? Присутствует ли в нём класс `face`?

`COCO` содержит 80 классов объектов, среди которых есть `person`, `car`, `bus`,
`dog`, `chair` и другие. Класса `face` в стандартном `COCO` нет.

4. Как изменить модель для детекции только лиц вместо людей?

Нужно использовать специализированную модель, обученную на датасете лиц,
например отдельную `YOLO`-модель для `face detection`, и передать её путь через
параметр `--model`, а также выбрать `--target-class face`.

5. Как изменить порог уверенности при вызове `model()`?

Порог уверенности задаётся параметром `conf`, например:

```python
results = model.predict(source=frame, conf=0.4, verbose=False)
```

В реализованной программе это настраивается через аргумент командной строки
`--conf`.

6. Как можно улучшить точность распознавания на видео с движущимися объектами?

- использовать более крупную модель, например `yolo11s.pt` или `yolo11m.pt`;
- увеличить размер входного изображения (`imgsz`);
- подобрать более подходящий `conf` и при необходимости `iou`;
- улучшить качество освещения и исходного видео;
- применять трекинг объектов между кадрами;
- дообучить модель на специализированных данных.
