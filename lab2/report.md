# Отчёт по лабораторной работе №2

## Краткое описание программы

В `weather.pl` реализована экспертная система рекомендаций по погоде для городов:
- Москва,
- Санкт-Петербург,
- Казань,
- Екатеринбург.

Совет формируется с учетом:
- температуры,
- погодного явления,
- ветра,
- рекомендованной активности.

Также выполнен **один проект на выбор** из списка: №3 «Поиск нарушений по автомобилю» (`project_car_search.pl`).

### Задание 2. Новое правило — учет ветра

Используются факты вида `weather/4` и правило `wind_advice/2`:

```prolog
wind_advice(Wind, 'Сильный ветер, будьте осторожны!') :- Wind > 10.
wind_advice(Wind, 'Ветрено, застегните куртку.') :- Wind >= 5, Wind =< 10.
wind_advice(Wind, 'Ветер слабый, комфортно.') :- Wind < 5.
```

Проверка:

```prolog
advice_for('Санкт-Петербург', A).
```

### Задание 3. Рекомендация активности

Добавлено правило:

```prolog
activity_advice(Temp, солнечно, 'Идеально для пляжа!') :- Temp > 20.
activity_advice(Temp, дождь, 'Можно посетить музей или кафе.') :- Temp >= 15, Temp =< 25.
activity_advice(Temp, снег, 'Хорошая погода для катания на лыжах.') :- Temp < 0.
activity_advice(_, _, 'Можно просто погулять по городу.').
```

Проверка:

```prolog
activity_advice(27, солнечно, A1).
activity_advice(18, дождь, A2).
activity_advice(-5, снег, A3).
activity_advice(10, облачно, A4).
```

## Выполнение 1 проекта на выбор

Выбран проект №3: **Поиск нарушений по автомобилю**.

Файл: `project_car_search.pl`.

Ключевые предикаты:

```prolog
find_violations_by_car(Car, Violations).
show_violations_by_car(Car).
```

Пример запросов:

```prolog
find_violations_by_car('А123ВС116', L).
show_violations_by_car('А123ВС116').
show_violations_by_car('М001ТТ116').
```

Пример вывода:

```text
L = [row('А123ВС116',speed_limit,202604011030),
     row('А123ВС116',red_light,202604031540)].

Нарушения для А123ВС116:
- Тип: speed_limit, Время: 202604011030
- Тип: red_light, Время: 202604031540

Для автомобиля М001ТТ116 нарушений не найдено.
```

## Ответы на контрольные вопросы

1. Что такое факт и правило в Prolog?
Факт — это безусловно истинное утверждение (например, `weather('Москва', 5, дождь, 7).`).
Правило — это условное утверждение вида `Голова :- Тело.`

2. Как выполнить запрос с переменной?
Использовать переменную в аргументе предиката, например:
`advice_for('Москва', A).`

3. Для чего используется предикат `forall/2` в программе?
Для выполнения действия для каждого решения условия.
В данной работе — для вывода советов по всем городам.

4. Как объединить несколько строк в одну в Prolog?
С помощью `atomic_list_concat/3`, например:
`atomic_list_concat([Part1, ' ', Part2], FullText).`
