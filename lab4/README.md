# Лабораторная работа №4

В папке находится автономная реализация лабораторной по теме:
`Создание баз знаний на основе открытых ресурсов` на материале открытых реестров ФИПС.

Используются:

- `Selenium` — навигация по дереву диапазонов в реестрах ФИПС;
- `BeautifulSoup4` — разбор HTML-карточек документов;
- `Pandas` — формирование сводных таблиц и CSV;
- подготовка `JSON`-файла для последующей загрузки в `Hadoop`.

## Что умеет скрипт

`main.py`:

1. Открывает раздел ФИПС:
   - `Реестр изобретений`
   - `Реестр полезных моделей`
2. Проходит по иерархии диапазонов номеров.
3. Находит конечные интервалы по 100 документов.
4. Скачивает HTML-карточки и собирает единый JSON.
5. Формирует CSV-сводки и краткий отчёт.

```bash
cd /Users/malikmuhametzanov/PycharmProjects/burn/lab4
../.venv/bin/python main.py --variant 1 --resume
```

## Полезные аргументы

- `--variant` — номер варианта из PDF (1–16).
- `--registry all|inventions|utility_models` — какой раздел собирать.
- `--limit` — ограничить число документов на раздел для тестового прогона.
- `--resume` — продолжать сбор с уже сохранённого промежуточного JSON и кэша.
- `--no-cache` — игнорировать локальный кэш карточек.
- `--headless / --no-headless` — запуск Chrome без окна или с окном.
